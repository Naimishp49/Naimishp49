# Minutes of Meeting (MoM) Documentation

Consistent meeting notes improve decision traceability and team alignment. Use the template below for every meeting.

## Template
```
Title: <Meeting Title>
Date: <YYYY-MM-DD>
Time: <Start–End> (<Timezone>)
Location/Link: <In-person/URL>
Facilitator: <Name>
Note-taker: <Name>
Attendees: <Names>

Agenda:
1. <Item>
2. <Item>

Discussion Summary:
- <Topic>: <Key points>

Decisions:
- <Decision> — Owner: <Name>

Action Items:
- [ ] <Action> — Owner: <Name> — Due: <YYYY-MM-DD>

Risks/Blockers:
- <Description>

References:
- <Links to docs, PRs, tickets>
```

## Example
```
Title: Sprint Planning
Date: 2025-08-22
Time: 10:00–11:00 (UTC)
Location/Link: Zoom
Facilitator: Priya
Note-taker: Alex
Attendees: Alex, Priya, Neeraj

Agenda:
1. Review previous sprint
2. Prioritize backlog

Discussion Summary:
- Velocity: 28 points last sprint
- Priorities: Authentication bugfix first, then dashboard chart

Decisions:
- Release 1.2 on Friday — Owner: Priya

Action Items:
- [ ] Fix OAuth callback error — Owner: Alex — Due: 2025-08-23
- [ ] Draft release notes — Owner: Neeraj — Due: 2025-08-24

Risks/Blockers:
- Possible staging outage during infra maintenance

References:
- JIRA-1234, PR-567
```

## Best Practices
- Keep MoM concise; focus on decisions and actions
- Assign a single owner per action with a due date
- Link to relevant issues/PRs for traceability
- Share notes within 24 hours
